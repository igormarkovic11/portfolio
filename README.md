# Personal Portfolio Website

[![React](https://img.shields.io/badge/React-19.2.4-61DAFB?style=flat&logo=react)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-8.0.4-646CFF?style=flat&logo=vite)](https://vitejs.dev/)
[![GitHub Pages](https://img.shields.io/badge/Deployed%20on-GitHub%20Pages-222222?style=flat&logo=github)](https://pages.github.com/)

A modern, responsive personal portfolio website built with React and Vite. Showcases projects, skills, and professional experience with a clean, minimalist design.

## ✨ Features

- **🎨 Modern Design** - Clean and professional interface
- **📱 Fully Responsive** - Works seamlessly on all devices
- **⚡ Fast Loading** - Optimized with Vite for lightning-fast performance
- **🎭 Smooth Animations** - Engaging user experience
- **📊 Projects Showcase** - Highlight your best work
- **💼 About Section** - Professional background and skills
- **📧 Contact Information** - Easy ways to get in touch
- **🎯 Skills Display** - Visual representation of technical skills

## 🛠️ Tech Stack

### Core
- **React 19.2.4** - UI library
- **Vite 8.0.4** - Build tool and dev server
- **JavaScript/JSX** - Programming language

### Styling & Icons
- **CSS3** - Custom styling
- **React Icons 5.6.0** - Icon library
- **Responsive Design** - Mobile-first approach

### Deployment
- **GitHub Pages** - Static site hosting
- **gh-pages 6.3.0** - Deployment automation

### Development Tools
- **ESLint 9.39.4** - Code linting
- **Vite Plugin React 6.0.1** - React fast refresh
